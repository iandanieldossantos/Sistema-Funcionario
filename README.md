👨‍💼 Sistema de Funcionário em Java

📌 Sobre o projeto
Aplicação desenvolvida em Java com foco em Programação Orientada a Objetos (POO), permitindo o cadastro de um funcionário e a atualização do seu salário com base em uma porcentagem informada pelo usuário.

O sistema também implementa uma regra de negócio que limita o aumento salarial a no máximo 30%.

🚀 Funcionalidades
Cadastro de funcionário (ID, nome e salário)
Exibição dos dados do funcionário
Aplicação de aumento salarial
Regra de validação: aumento máximo de 30%
Atualização automática do salário

🛠️ Tecnologias utilizadas
Java
Programação Orientada a Objetos (POO)
▶️ Como executar o projeto
Clone o repositório:
git clone https://github.com/iandanieldossantos/Sistema-Funcionario.git
Abra o projeto em uma IDE (IntelliJ, Eclipse ou VS Code)
Execute a classe Program.java

💻 Exemplo de execução
ID: 1
Nome do funcionario: Ian
Salario: 2000.00

Dados:
ID 1, Nome: Ian, Salario atualizado: $ 2000.00

Porcentagem de aumento: 10

Dados atualizados:
ID 1, Nome: Ian, Salario atualizado: $ 2200.00
📚 Conceitos aplicados
Classes e objetos
Encapsulamento (atributos privados + getters/setters)
Métodos com regras de negócio
Validação de dados
Entrada de dados com Scanner
🧠 Regra de negócio implementada

O sistema permite aumento salarial apenas entre 0% e 30%, garantindo maior controle e evitando valores inválidos.
